Iguana

Iguana is an open source issue management system with a kanban board, sprint planning
and timelogging features, and anything else you need to make working on a project in groups pleasant.

You can get in on github.

Search function

Whenever you don't remember which syntax the search function has you can click the help where you also will be redirected to, in case of a search-request that produced an empty result.

Search example regex

There are way to many tools out there that are usable in multiple ways but don't provide the functionality to use regex for search functionalities so they are missing a more complex way to do so. In this example we search for Projects with a description that contains the word demo, which is true for the result because demonstration is part of the description there. You can also note the history of previous searches in the search-bar.

Share search expressions

Next to those list of previous searches iguana allows you to share your search expressions for a specific project to share common searches easily with other members of your team. In private state all of our expressions aren't saved permanently yet, so we save the one we want to share. After that we are able to share this expression with some projects. In this case we rename the filter to priority==3 && number< 9 and share it with our demonstration project, where the filter indeed appears and produces the desired output.
